XROOTD Stats
================


Install
----------------

Copy xrootd_stats.py from python_modules to your python modules directory, e.g. :

- /usr/lib/ganglia/python_modules
- /usr/lib64/ganglia/python_modules

Copy xrootd_stats.pyconf to the gmond conf.d directory, e.g. :

- /etc/ganglia/conf.d/
